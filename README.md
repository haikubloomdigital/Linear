# Website Linear Creative

Website profesional untuk Linear Creative - Percetakan Online & Jasa Desain Kreatif di Jogja.

## Fitur
- Desain modern, bersih, dan responsif
- Optimasi SEO untuk "Percetakan Jogja"
- Konversi tinggi ke WhatsApp
- Ringan & cepat loading
- Kompatibel dengan semua device

## Teknologi
- HTML5 semantik
- CSS3 dengan variabel dan flexbox/grid
- JavaScript Vanilla (tanpa library/framework)
- Hosting gratis di GitHub Pages

## Cara Deploy ke GitHub Pages
1. Buat repository baru di GitHub dengan nama `linear-creative`
2. Upload semua file ke repository
3. Pergi ke Settings > Pages
4. Pada Source, pilih branch `main` dan folder `/root`
5. Klik Save
6. Website akan live di: `https://username.github.io/linear-creative/`

## Customization
1. Ganti gambar placeholder dengan gambar asli
2. Update nomor WhatsApp di semua link
3. Update link media sosial
4. Tambahkan logo klien asli
5. Update meta description sesuai kebutuhan

## Catatan untuk Pemilik Bisnis
- Website ini GRATIS dan open source
- Dapat dihosting di GitHub Pages tanpa biaya
- Semua konversi langsung ke WhatsApp
- SEO friendly untuk pencarian lokal Jogja
- Konten dapat diubah sesuai kebutuhan
